# Music Player with Keyboard Controller

This project is a simple music player built with Pygame.

## Features
- Play current track
- Stop playback
- Switch to next track
- Return to previous track
- Show current track name
- Display playback status
- Show current position and progress bar

## Keyboard Controls
- `P` = Play
- `S` = Stop
- `N` = Next track
- `B` = Previous track
- `Q` = Quit

## Project Structure
- `main.py` — entry point
- `player.py` — player logic
- `music/` — audio files
- `README.md` — project description
